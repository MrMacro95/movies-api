# Movies API

A simple REST API built using Node.js, Express, and MongoDB.

## Features
- Create a movie
- Get all movies
- Get movie by ID
- Update movie
- Delete movie

## Tech Stack
- Node.js
- Express
- MongoDB (Mongoose)

## How to Run

1. Install dependencies  
   npm install  

2. Start server  
   node index.js  

3. Server runs on  
   http://localhost:8005
